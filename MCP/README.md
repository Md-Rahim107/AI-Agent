## MCP

Turns Telegram into a smart personal assistant using two dedicated MCP Servers: Google Calendar MCP (create, delete, get, recurring events with conflict detection) and Google Sheets MCP (event logging). Powered by Groq/Gemini with conversation memory, it handles scheduling, recurring meetings, and real-time replies — all in Bangladesh time.
