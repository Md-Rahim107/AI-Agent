## Restaurant Chat Bot
A public chat AI Agent powered by Groq and Gemini with conversation memory. Guides users through menu info, restaurant details, or step-by-step table reservations. Checks Google Sheets for existing bookings by email, appends new ones, and sends polished confirmation emails via Gmail when the user opts in.
