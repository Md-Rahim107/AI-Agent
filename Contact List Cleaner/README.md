## Contact List Cleaner
Pulls contacts from Google Sheets, removes duplicates, and verifies each email using the Emailable API in a controlled loop. Only deliverable addresses are kept. The sheet is cleared and repopulated with clean data. Personalized confirmation emails are then sent to all verified contacts via Gmail — fully automated from start to finish.
