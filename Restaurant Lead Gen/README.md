## Restaurant Lead Gen
Users message a location and the AI Agent searches via Google Serper API, extracting restaurant name, ratings, address, hours, website, and phone. Prioritizes 4.0+ rated results, validates output, and appends to Google Sheets. An Error Workflow logs failures and notifies via Gmail automatically.
