## Intelligent Customer Support System
 
n8n workflows create an intelligent customer support system. Documents are ingested into Qdrant via Google Gemini embeddings. The support bot monitors Gmail, classifies tickets, retrieves context from the vector store, generates professional replies, and tracks sentiment in real-time — with escalation and Telegram alerts built in.
